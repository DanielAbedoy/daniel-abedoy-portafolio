Este proyecto fue el más bonito que he realizado, ya que el proceso del la creacion del hardware fue lo más complicado que se puede imaginar, claro sin experiencia y sin herramientas necesarias, despues de muchos intentos se logró realizar y funciona al 100%. El control esta programado con el microcontrolador Atmega328p con Arduino en este caso el modelo Nano. Este control puede manipular más hardware simpre y cunado tenga la antena receptora NRF24L01
